# PERN_Todo

A full stack ToDo list application built using the PERN stack (PostgreSQL, Express.js, React, Node.js).

## Features

- Add, update, delete, and view todos
- Fully RESTful API
- Connected frontend and backend
- PostgreSQL database

## Tech Stack

- PostgreSQL
- Express.js
- React
- Node.js

## Setup

### Backend
```bash
cd server
npm install
node index.js
